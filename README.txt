== Description ==

**WITH MWB CF7 INTEGRATION WITH ENGAGEBAY, CONNECT CF7 TO ENGAGEBAY AND SEND ALL YOUR DATA ( SUBMISSIONS/ENTRIES) AUTOMATICALLY OVER THE ENGAGEBAY.**

**MWB CF7 Integration with Engagebay** integrates CF7 with Engagebay. This plugin sends all the data of Contact Form 7 (submissions/entries) over Engagebay as per the modules available in the Engagebay. You can associate any field type using the custom fields with this Engagebay add-on for Contact Form 7.

Contact Form 7 is one of the best tools with which you can easily create lead generating forms, manage multiple forms, and customize them. On the other hand Engagebay is all-in-one CRM, Sales, Marketing, and Support software. This plugin is a combination of these two which will definitely increase your business fecundity and growth. 

The Enagagebay database will automatically update as visitors submit the forms on your site. In this manner- you can attract more leads, grow your customer engagement, and increase your  revenue.

== WHAT IS INCLUDED IN FREE  CF7 INTEGRATION WITH ENGAGEBAY  PLUGIN == 

**1)** **Integrate CF7 With Engagebay**
Enter your API credentials (API key, Base URL) to connect Engagebay and Contact Form 7. On the plugin’s dashboard, you can view the connection status, owner name, and synced contact forms. Additionally, you can disconnect and reauthorize this plugin whenever you want.


**2)** **Associate Form Fields with Engagebay Modules**
You can create CF7 forms and associate form fields with Engagebay modules. You can effortlessly sync form submissions/entries data over an Engagebay account.


**3)** **Filter CF7 submissions As Per Feeds**
You can filter the Contact form 7 submissions based on user input using AND/OR logic. This logic will filter CF7 forms submissions and send it to Engagebay depending on user inputs.


**4)** **Comprehensive Log**
MWB CF7 Integration with Engagebay plugin will provide a detailed log of each Contact Form 7 entry sent to Engagebay. Integrate your Contact Form 7 with Engagebay that comes with logging of all the API interaction with Engagebay for better error handling.

**5)** **Delete Plugin Data On Uninstall**
This plugin allows you to delete a huge amount of data stored in the database whenever you uninstall it.

**6)** **Receive Error Email Notification**
MWB CF7 Integration With Engagebay sends email notification if any input error occurs in the process of data, sending entries over Engagebay. You will be notified whenever any minor error occurs.

**7)** **Engagebay Custom Values**
You can opt for custom values from the field type for field mapping of feed. Then, your value will sync. Also, you can name the custom value and select the CF7 field as per your requirement.

== Installation ==

= Automatic installation =
Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of the plugin, log in to your WordPress dashboard, navigate to the Plugins menu, and click Add New.

In the search field type **"MWB CF7 Integration with Engagebay"** and click Search Plugins. Once you've found our MWB CF7 Integration with Engagebay plugin you can view details about it such as the point release, rating, and description. Most importantly, of course, you can install it by simply clicking **"Install Now"**.

= Manual installation =
The manual installation method involves downloading our MWB CF7 Integration with Engagebay  and uploading it to your web server via your favorite FTP application. The WordPress codex contains [**instructions on how to do this here**](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

= Updating =
Automatic updates should work like a charm; as always though, ensure you backup your site just in case.

== Frequently Asked Questions ==

= How can I manage my submissions data with MWB CF7 Integration with Engagebay? =

MWB CF7 Integration with Engagebay integrates Contact Form 7 with Engagebay which helps you send all the form submissions data over Engagebay according to the form field mappings

= How is Engagebay CRM supportive? =

Yes you will get a full detailed log from where you can see all the data which was sent as request for particular action. To see this we have given one option of Logs Tab.

= Can I get the error notification mail on my personal id by this plugin? =

By default, this plugin sends the error notification mail on the admin email. But, if you want you can enter your desirable id for receiving the error notifications..

= Where can I view the entire submission details in this plugin? = 
This plugin provides a log where you can see all your submission data as well as the response generated from Engagebay. Here, you can view enagagebay objects, enagagebay ID, event, timestamp, request, etc.

= Can I map my fields with Engagebay custom values in this plugin? =
Yes, for field mapping of feed you can easily choose the custom value from the plugin’s field type option. Also, you can name the custom value according to your choice.


= Can I add desirable conditions for sending the submissions data over Engagebay? =
Yes, MWB CF7 Integration with Engagebay provides the option- Conditional Filters. With the help of it, you can apply conditional filters on your form submissions to add your desirable conditions.
